# Signed Off

**There is done and there is DONE. Signed Off ensures every 
piece of work is DONE right.**

Signed Off is a GitHub App that brings structured, enforceable 
checklists to every GitHub issue — automatically. Define your 
standards once, apply them everywhere, and ensure nothing closes 
until it's actually done.

## What it does

- ✅ **Auto-posts checklists** on every issue, matched by type
- ✅ **Org-wide templates** defined once, applied across every repo
- ✅ **Closure enforcement** — Advise, Warn, or Block per template
- ✅ **Exception approvals** — team-gated, no self-approval
- ✅ **Live SVG progress badge** embeddable anywhere
- ✅ **Zero infrastructure** — state lives inside GitHub itself
- ✅ **Smart Defaults with Customizability** — Install and just start, and tune to your needs
